Integritetspolicy

Din integritet är viktig för oss på Lokomotiv Städ & Fönsterputs AB. Här förklarar vi tydligt vilka uppgifter vi samlar in, varför vi gör det och hur du kan kontakta oss om du vill veta mer.

Till kontaktformuläret Tillbaka till startsidan

Vilken information samlar vi in?

När du kontaktar oss via formuläret kan vi samla in:

  • Namn
  • E-postadress
  • Telefonnummer
  • Vald tjänst eller valda tjänster
  • Meddelande och övrig information du själv skriver

Varför samlar vi in information?

Uppgifterna används för att kunna svara på din förfrågan, kontakta dig och ge offert. Den rättsliga grunden är att kunna hantera din begäran innan ett eventuellt avtal.

Hur skickas och lagras informationen?

Kontaktformuläret skickas via Formspree. Uppgifterna sparas endast så länge de behövs för att hantera din kontakt eller offertförfrågan, om vi inte enligt lag behöver spara uppgifterna längre.

Analytics

Webbplatsen kan använda enkel webbstatistik via Plausible Analytics för att förstå hur webbplatsen används. Statistiken används på övergripande nivå och är inte avsedd att identifiera enskilda besökare.

Delar vi din information?

Vi säljer aldrig dina personuppgifter. Uppgifter kan däremot behandlas av tekniska leverantörer som hjälper oss att driva webbplatsen och kontaktformuläret.

Dina rättigheter

Du har rätt att begära tillgång till, rättelse av eller radering av dina personuppgifter. Du kan också invända mot eller begära begränsning av behandling.

Kontakt

Har du frågor om integritet eller vill utöva dina rättigheter? Kontakta oss på eric@lokomotivstad.se.
